Windows 实用脚本集 (Windows Utility Scripts)

这里收集了几个用于提升 Windows 日常操作效率的轻量级小脚本。无需复杂的安装与配置，双击即可解决高频痛点。

🛠️ 包含的工具

1. 一键关闭系统代理 (关闭代理.pyw)

在某些网络环境下（如经常使用代理软件、VPN 异常崩溃等情况），Windows 的系统代理经常会被意外打开，导致浏览器无法正常上网。

功能：通过直接修改注册表强制关闭 Windows 系统代理，并调用底层 API 立即刷新网络设置使其生效。

特点：采用 .pyw 后缀，纯静默运行。双击后没有任何黑框闪烁，没有任何弹窗打扰，在无感中瞬间修复网络。

环境要求：需要安装 Python 3.x 环境。

2. 一键打开自启动目录 (开机启动.bat / windows自启动目录.pyw)

想要让某个软件开机自启，却总是记不住那串又长又难找的隐藏路径（C:\Users\用户名\AppData\Roaming\...）？

功能：利用 Windows 原生 shell:startup 协议，一键直达当前用户的开机自启动文件夹。

特点：极度轻量。将需要开机自启的程序快捷方式直接拖入打开的文件夹即可生效。

环境要求：.bat 版本双击直接可用，无需任何环境。

🚀 进阶玩法：解决“开机没网络”的终极痛点

场景痛点：用完代理忘记关闭，电脑直接关机。下次开机时，由于代理设置残留，发现电脑完全没有网络，必须手动去系统设置里找半天才能关掉。

完美解决方案（开机自动重置网络）：
我们可以将上述两个脚本结合使用，让电脑每次开机时，自动、静默地帮你清理一次代理残留，确保开机即有网。

双击运行 开机启动.bat，系统会弹出一个文件夹（这就是你的开机自启动目录）。

找到 关闭代理.pyw 脚本。

右键点击 关闭代理.pyw -> 选择 “发送到” -> “桌面快捷方式”。

将桌面刚刚生成的快捷方式，直接拖入刚才打开的自启动文件夹中。

🎉 搞定！ 以后每次电脑开机，系统都会在后台悄悄执行一次“关闭代理”的操作。因为它没有任何黑框和弹窗，所以你完全感受不到它的存在，但你再也不会遇到“开机没网”的尴尬情况了。

⌨️ 快捷键绑定技巧

如果你希望在日常使用中也能随时一键切断代理，可以配合 Windows 快捷键：

将 关闭代理.pyw 创建一个快捷方式放到桌面。

右键点击该快捷方式 -> 属性。

在“快捷键”一栏中，按下你想设置的按键组合（例如 Ctrl + Alt + P）。

点击“确定”。

当你的网络突然因代理崩溃中断时，只需盲按 Ctrl + Alt + P，网络瞬间恢复畅通！
